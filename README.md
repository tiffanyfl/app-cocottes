# app-cocottes

## Getting started

1. Install cordova

```
npm install -g ionic cordova
```

2. Install all node modules

```
npm install
```

3. Create ```www``` and ```build``` folder using this command :

```
ionic serve
```

4. Install ```platforms``` and ```plugins```

```
ionic cordova prepare
```


## Use ionic

1. Launch server

```
ionic serve
```
